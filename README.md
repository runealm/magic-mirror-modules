# magic-mirror-modules

Custom modules for [MagicMirror²](https://magicmirror.builders/) built for club and venue displays.

## Modules

### [MMM-KioskPrices](./MMM-KioskPrices)
Displays a simple price list read from a local JSON file. Perfect for sports clubs, canteens or any venue with a kiosk. Prices update automatically without restarting MagicMirror.

### [MMM-Ordensmann](./MMM-Ordensmann)
Displays a monthly duty roster ("ordensmann") read from a local JSON file. Shows the current and next month automatically based on the system date.

---

## Requirements

- MagicMirror² v2.15+
- Node.js v18+

## Installation

Clone or download the module you want into your MagicMirror `modules` folder:

```bash
cd ~/MagicMirror/modules
git clone https://github.com/runealm/magic-mirror-modules
```

Then copy the module folder you want into your `modules` directory and follow the individual README for configuration.
